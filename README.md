# AS73211-Colorimeter-STM32F7

## About project

This is a demo project for [ams AS73211](https://ams.com/as73211) XYZ color sensor using [COLOR 6 Click](https://www.mikroe.com/color-6-click) board from Mikroelektronika. Project was built using following ecosystem:

 - [mikromedia+ for STM32F7](https://www.mikroe.com/mikromedia-4-stm32f7) development board by Mikroelektronika
 - STM32CubeMX - initialization code generator by STMicroelectronics
 - System Wokrbench for STM32
 - mikroBootloader – Free and simple application by Mikroelektronika for programming MCU with a preloaded USB-HID bootloader.
 
More info can be found [here](https://www.optolab.ftn.uns.ac.rs/index.php/education/project-base/267-as73211-colorimeter)
